# Monorepo

## Usage

1. Clone template

```
npx degit rjschie/templates/monorepo <project> && cd $_
```

2. Initialize project

```
git init .
yarn
yarn setup
```

3. Initial commit

```
git add . && git commit -m "chore: initial commit"
```

4. Add Origin

```
git remote add origin git@github.com:rjschie/<project>
```
